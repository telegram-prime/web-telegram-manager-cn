# WBTG – Web Telegram Manager

**Telegram Web 多账号桌面管理工具**

一款强大的桌面应用程序，可在一个统一界面中管理、组织、监控和操作多个 Telegram Web 账号。

WBTG 将混乱的浏览器标签页转变为干净、专业且可靠的 Telegram Web 多账号工作空间。

---

### 🌟 为什么选择 WBTG？

无需手动打开数十个浏览器配置文件，WBTG 基于 Web 配置文件创建并管理 **隔离的 Telegram Web 会话**。每个会话拥有独立的持久化存储、代理设置、指纹及任务状态。

### ⚙️ 核心功能

| 分类                       | 功能 |
|----------------------------|------|
| **多账号管理**             | 在一个程序中加载、整理和恢复数十个 Telegram Web 账号 |
| **独立配置文件**           | 每个账号使用独立的持久化配置文件目录 |
| **无需重新输入 2FA 的会话导入** | 将现有 Telegram 会话导入程序时，无需再次输入 Telegram 两步验证密码 |
| **可携带工作区**             | 可将整个程序文件夹连同配置与账号资料一起迁移到新位置，而无需重新绑定路径 |
| **收件人进度追踪**           | 将已发送、待处理和失败的收件人分开记录，便于中断恢复、审计和重试 |
| **灵活的对话打开模式**       | 提供多种收件人对话打开方式，以更好兼容不同 Telegram Web 流程 |
| **账号监控**               | 实时显示未读消息、冻结账号和已登出状态 |
| **智能发送器**             | 支持文本、媒体和机器人指令发送，带延迟、限额和队列追踪 |
| **媒体发送流程**           | 完整的图片发送工作流，支持标题、文件夹选择和顺序/随机模式 |
| **任务计划**               | 内置计划任务系统，支持延期和取消 |
| **代理管理**               | 批量分配代理（支持多行列表），可顺序或随机分配 |
| **日志与报告**             | 实时详细日志、可导出报告及每个账号的发送总结 |
| **环境自定义**             | 每个账号独立设置缩放、指纹、语言和时区 |
| **会话持久化**             | 程序重启后自动恢复所有会话 |
| **快速客服支持**           | 提供高响应支持渠道，用于处理配置、授权和运行中的问题 |
| **自动更新**               | 内置版本检查功能，并提供最新程序版本的直接下载链接 |

### 🚀 核心优势

- **完全隔离** — 每个账号拥有独立的配置文件目录  
- **专业发送流程** — 高级发送模块，支持进度追踪和中断恢复  
- **持久化工作空间** — 所有设置和数据在重启后自动保存  
- **便捷监控** — 一目了然地查看每个账号的状态  
- **灵活计划** — 可定时执行任务，无需持续人工干预
